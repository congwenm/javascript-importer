# Javascript Importer Package

This package is for convenience so the developer would not have to scroll up and down a javascript page to import Modules.

## Usage

Press:

`Ctrl`-`Opt`-`i`

Enter:

`Model from 'src/model';`

it will get inserted at top of the file.
